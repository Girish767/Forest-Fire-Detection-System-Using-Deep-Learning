How It Works

1. Input Processing

Upload an image or video. The system automatically resizes and normalizes the visual data using the SigLIP processor to match the model's requirements.

2. AI Analysis

Our fine-tuned SigLIP (Vision Transformer) model scans the input. It uses attention mechanisms to focus on fire and smoke patterns, distinguishing them from clouds or fog.

3. Real-time Results

Get instant classification: Normal, Smoke (Early Warning), or Fire (Moderate/Heavy). Confidence scores help you assess the risk.

Model Architecture

  • Base: Google SigLIP (Sigmoid Loss for Language Image Pre-training)
  • Training: Fine-tuned on a custom dataset of 1000+ fire/smoke images.
  • Optimization: AdamW optimizer with learning rate scheduling.
